## AggmUI examples

To quickly get started with AggmUI, you can run `aggmui.py` directly, or `example.py` .

### Code example

```python

from aggmui import *

view = View(" Window Title", 400, 600)
view.spacer()
view.heading("Heading")
view.spacer()
view.text("Text")
view.spacer()
view.mainloop()

```
